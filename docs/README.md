# CS Docs

CS 학습 문서는 주제별로 나눠 관리합니다.

| 폴더 | 내용 |
|------|------|
| `01_os/` | 운영체제, 프로세스/스레드, 메모리, syscall |
| `02_network/` | OSI/TCP-IP, DNS, HTTP/HTTPS, CDN, NAT, 로드밸런싱 |
| `03_database/` | SQL, 트랜잭션, 락, NoSQL, 캐싱 |
| `04_distributed-systems/` | CAP, 메시징, 분산 시스템 기본기 |
| `05_system-design/` | 확장성, 안정성, 백업, CI/CD, 부하 처리 |
| `06_security/` | 인증/인가, 보안 기본기 |
| `07_observability/` | 모니터링과 로깅 |
| `08_jvm/` | JVM 구조, 메모리, GC |
| `09_container/` | Docker, Kubernetes 기본기 |
| `deepdive/` | 커널, TCP, 프록시, 스토리지, 관측, 성능 심화 |
| `agents/` | AI 전문 에이전트 프롬프트 |
| `rules/` | 문서 작성과 CS 문서 규칙 |
| `templates/` | 재사용 문서 템플릿 |

실습 자산은 `../ops/README.md`를 참고합니다.
