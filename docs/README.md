# cs-basic 문서

이 디렉토리는 주제별 번호 체계로 문서를 정리합니다. 앞 번호는 학습과 운영 흐름을 나타내며, 90번대는 저장소 관리용 문서입니다.

## 문서 구조

- [01-os](./01-os/): 운영체제 - 프로세스, 메모리, 커널, 시스템 콜 등 운영체제 기본 문서를 모읍니다.
- [02-network](./02-network/): 네트워크 - OSI/TCP-IP, DNS, HTTP, 로드밸런싱 등 네트워크 기본 문서를 모읍니다.
- [03-database](./03-database/): 데이터베이스 - SQL, 트랜잭션, NoSQL, 캐싱 문서를 모읍니다.
- [04-distributed-systems](./04-distributed-systems/): 분산 시스템 - CAP, 메시징 등 분산 시스템 기본 문서를 모읍니다.
- [05-system-design](./05-system-design/): 시스템 디자인 - 확장성, 안정성, 백업, 배포 설계 문서를 모읍니다.
- [06-security](./06-security/): 보안 - 인증과 보안 기본 문서를 모읍니다.
- [07-observability](./07-observability/): 관측성 - 모니터링과 로깅 기본 문서를 모읍니다.
- [08-jvm](./08-jvm/): JVM - JVM 구조, 메모리, GC 기본 문서를 모읍니다.
- [09-container](./09-container/): 컨테이너 - Docker와 Kubernetes 기본 문서를 모읍니다.
- [10-deep-dive](./10-deep-dive/): 심화 주제 - 커널, TCP, 스토리지, Kubernetes 등 심화 문서를 모읍니다.
- [90-standards](./90-standards/): 문서 작성 표준 - 저장소 문서 작성 규칙과 운영 기준을 관리합니다.
- [91-templates](./91-templates/): 문서 템플릿 - 반복해서 사용하는 문서 템플릿을 관리합니다.
- [99-agents](./99-agents/): AI 에이전트 가이드 - CS 학습과 면접 준비에 활용할 에이전트 역할 정의를 관리합니다.
