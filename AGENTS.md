# AGENTS.md — cs-basic Codex 작업 지침

이 저장소는 CS 기본기와 심화 주제를 정리한 지식 베이스입니다. Codex 작업 시 `CLAUDE.md`와 `docs/rules/`의 규칙을 동일하게 따릅니다.

## 공통 원칙

- 학습 문서는 `docs/` 아래에 둡니다.
- 실습 코드나 검증 스크립트는 `ops/` 아래에 둡니다.
- 설명은 원리, 예시, 실무 적용 포인트를 함께 포함합니다.
- 네트워크/OS/DB/분산 시스템 내용은 서로 연결되는 지점을 명확히 표시합니다.

## Claude와의 싱크

- `CLAUDE.md`는 Claude용 상세 지침입니다.
- `AGENTS.md`는 Codex용 진입점입니다.
- 공통 규칙은 `docs/rules/`를 기준으로 유지합니다.

## 작업 체크리스트

- 기존 변경 확인
- 링크 검사
- 코드 예제가 있으면 해당 언어의 기본 문법 검사
- `git diff --check` 수행
